About This App

This app demonstrates a machine learning model trained to classify human physical activities using smartphone sensor data. The dataset is sourced from the UCI Machine Learning Repository and contains recordings from 30 volunteers performing 6 activities — Walking, Walking Upstairs, Walking Downstairs, Sitting, Standing, and Laying — while wearing a Samsung Galaxy S II on their waist.


Each sample in the dropdown represents one 2.56-second window of sensor data, described by 561 features extracted from the phone's accelerometer and gyroscope. These are real unseen test samples — people the model was never trained on. Select any sample and click Predict to see how the model classifies that activity.
